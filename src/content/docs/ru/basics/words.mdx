## Окончания

| Часть речи      | Женский | Мужской | Средний | Множественное |
| --------------- | ------- | ------- | ------- | ------------- |
| Местоимение     | -       | -       | -       | i             |
| Существительное | а       | о       | -       | ы             |
| Глагол          | тä      | тö      | тü      | тï            |
| Прилагательное  | нä      | нö      | нü      | ны            |
| Наречие         | е       | е       | е       | е             |

## Местоимение

### Склонение

| Значение              | Местоимение         | Множественная форма |
| --------------------- | ------------------- | ------------------- |
| Я                     | Я                   | (Мы) Яi             |
| Ты / Вы (уваж. форма) | Ню                  | (Вы) Нюi            |
| Он/Она                | Не + окончание сущ. | (Они) Неi           |

! | У личный местоимений "i" читается как "ий", почти как "й".

## Глагол

### Время

В ферийском для указания времени нужно добавить перед или после глагола необходимый предлог:

| Прошлое (до глагола) | Настоящее | Будущее (после глагола) |
| -------------------- | --------- | ----------------------- |
| бi (было)            | -         | дi (будет)              |
